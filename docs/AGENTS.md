# AGENTS.md

这是一个 Jetson/OpenCV 学习项目，内容是视觉识别目标，然后控制云台舵机追踪目标。

## Rules

* 最重要的一点:没有我明确的命令不准修改代码
* 每次只做一个小任务，优先最小改动。
* 不要主动大规模重构，不要引入复杂架构或新增依赖，除非我明确要求。
* 如果我只是要求解释、检查代码或分析报错，不要修改文件。
* 如果需要改代码，先简短说明修改计划，再修改。
* 除非我明确要求完整代码，否则优先给核心片段或局部修改。
* 新增或修改代码要符合项目现有风格，保持简单直观，方便学习理解。
* 关键逻辑、函数和配置项需要写简短中文注释；显而易见的语法不要重复注释。
* 修改后简短说明改了哪里、为什么改，以及如何验证。

## Ignore

不要分析 `.venv`、`__pycache__`、`.git`、`build`、`dist`、`runs`、`datasets`、`weights`、`models`，除非我明确要求。

## Run

```bash
python main.py
```
